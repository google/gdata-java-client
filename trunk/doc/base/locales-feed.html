Locales Feed

This read-only feed defines the permitted locales for Google Base. The locale value identifies the language, currency, and date formats used in a feed.

Contents

  1. Feed URL
  2. Usage example

Feed URL

Typically, you will simply get the entire locales feed, but you can optionally specify a single locale. 

http://www.google.com/base/feeds/locales/<locale>

locale limits the feed to return information for a single location. Currently, the following locales are defined:

United States en_US
Great Britain en_GB
Germany de_DE

This URL lists the entire locales feed: 

http://www.google.com/base/feeds/locales

This URL obtains only those attributes defined for the US locale:

http://www.google.com/base/feeds/locales/en_US

You can also run batch queries on the locales feed.

There is a link in the response that points to the item types feed where more data can be found about this locale. You can use this information to write code that automatically takes you to the next level of detail. 

<link rel="related" type="application/atom+xml" href="http://www.google.com/base/feeds/itemtypes/en_US" title="Item types in en_US"/>
